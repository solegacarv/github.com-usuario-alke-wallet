Alke Wallet es más que una billetera digital: es una experiencia de usuario diseñada para ser simple, visual y confiable.
Este proyecto combina HTML, CSS, Bootstrap y jQuery para ofrecer una interfaz moderna y responsiva, donde cada pantalla refleja un flujo real de uso: desde el login hasta las transacciones.
El propósito de Alke Wallet es demostrar cómo una aplicación web puede ser modular, escalable y mantenible, cuidando tanto la estética como la usabilidad. 
Es un proyecto pensado para aprender y aplicar buenas prácticas de desarrollo, al mismo tiempo que sirve como portafolio profesional.
